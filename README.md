# Cat Energy

Адаптивный лендинг для бренда функционального питания котов. Проект сверстан по методологии Desktop First.

## Демо
Ссылка на GitHub Pages:
https://ksaiko-vlad.github.io/Cat-Energy/

## Особенности реализации

### Адаптивная верстка
В проекте использован подход Desktop First. Для адаптации под разные устройства применены медиазапросы с брейкпоинтами:

- **max-width: 1180px** 
- **max-width: 767px**
- **max-width: 692px**

Вместо фиксированных значений для многих элементов (шрифты, отступы, размеры блоков) использованы относительные единицы и проценты, что обеспечивает корректное отображение на промежуточных разрешениях.

### Изменения для адаптивов
Вносились изменения размеров и положения элементов

## Технологии
- HTML5
- CSS3 (Flexbox, Grid, медиазапросы)
- JavaScript (слайдер для блока "Живой пример")
- Яндекс.Карты использовался iframe из-за невозможности получить Ключ разработчика Яндекс

## Запуск проекта
1. Склонировать репозиторий
2. Открыть index.html в браузере (рекомендуется использовать Live Server)
